class openaerostruct.structures.fem.FEM(**kwargs)[source]¶

Bases: openmdao.core.implicitcomponent.ImplicitComponent

Component that solves a linear system, Ax=b.

Designed to handle small, dense linear systems (Ax=B) that can be efficiently solved with sparse lu-decomposition. It can be vectorized to either solve for multiple right hand sides, or to solve multiple linear systems.

A is represented sparsely as a local_stiff_transformed, which is an ny x 12 x 12 array.
